Experiences Projects

Software Engineering Intern

Kabbage, Inc. Atlanta, Georgia

June 2019 - August 2019

At Kabbage, I worked with senior software engineers to develop the Kabbage backend data platform. I was very lucky to work on a real product and my contribution was also deployed into the production environment.

During the internship, I learned how to make me comfortable to work on a large codebase and practiced technologies, such as microservices, writing maintainable code, RESTful API and database model design, that I learned from school. Most importantly, working in a collaborative environment. Code review to give and receive suggestions from each other; working with QA to debug and test new functionalities; joining biweekly sprint to learn our progress and next plan. All of these activities trained my professional skills and jump-started my career.

Undergraduate Research Assistant

Distributed Computing Systems Group, UMN

January 2018 - May 2018

I was supervised by Professor Jon Weissman and Professor Abhishek Chandra, and worked with a Ph.D. student, Kwangsung Oh, on the Wiera project , which is a policy-driven distributed key-value storage system. In this project, my role was to design new response classes, write policies to build the different consistency models, test and debug the Wiera system. This experience highly improved my code analysis and debugging abilities. Meanwhile, I also learned the architecture of a real distributed storage system and the effects of different consistency models.
